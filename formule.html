Kosinusna teorema:

Kosinusna teorema povezuje odnos stranica i njihovih suprotnih uglova i to na osnovu geometrijske cinjenice da u istom trouglu:

*Veci ugao "gleda" vecu stranicu, a manji ugao "gleda" manju stranicu*

a2 = b2 + c2 - 2bc · cosα

b2 = a2 + c2 - 2ac · cosβ

c2 = a2 + b2 - 2ab · cosγ

-Ako je ugao γ = 90° = 0 onda je cos 90° - 0 i teorema se svodi na:

c2 = a2 + b2

T.j pitagorinu teoremu.
